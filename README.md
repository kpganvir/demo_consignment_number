# NFT Marketplace

## Technology Stack & Tools

- SpringBoot
## Requirements For Initial Setup
- Install java 1.8

## Setting Up
### 1. Clone/Download the Repository 

### 2. run application
```
$ cd demo
$ open command window cmd.exe
$ mvn clean install 
$ mvn spring:run 
```


### 4. test api
